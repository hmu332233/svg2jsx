## svg2jsx

- svg 파일들을 jsx파일들로 변환시켜준다.

### 사용법
1. 프로젝트를 받는다.
```
$ git clone https://github.com/hmu332233/svg2jsx.git
```
2. `images`폴더에 svg 파일들을 넣는다.
3. `$ node start` 실행
```
$ node start
```
4. `results`에 변환된 jsx 파일들이 생성되어 있다.
